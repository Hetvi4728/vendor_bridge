# 📌 About The Project

VendorBridge is a centralized ERP solution built to simplify procurement and vendor management operations for organizations.

The platform digitizes the entire procurement lifecycle including:

- Vendor onboarding
- RFQ management
- Quotations
- Purchase orders
- Invoice tracking
- Approval workflows
- Procurement analytics
- Real-time status tracking

The goal of VendorBridge is to reduce manual procurement inefficiencies through automation, transparency, and secure role-based workflows.
